# Code-Crafters
## Problem Statement :
 Green Energy: "Phantom Power" Cost Estimator
 The Problem: People are often unaware of how much money and energy they waste through "phantom loads"—
devices that draw power even when turned off or in standby mode (like microwaves, TVs, or plugged-in chargers).
The Challenge: Create a calculator that visualizes the environmental and financial cost of idle appliances.
Deliverables:
• Build an interactive calculator where a user can select standard household appliances from a dropdown menu
and input their local electricity rate (₹/kWh).
• Calculate the estimated annual cost and carbon footprint (in kg of CO2) of leaving these devices plugged in
24/7.
• Include a data visualization (like a pie chart or bar graph) showing which appliance is the biggest "energy
vampire."
Domain of Project: Web Development / Data Visualization
